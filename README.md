# Patient Health Database

Author: David Ward

## Purpose
The code in this repository will run a patient 
health database.  
The code is written in Python.

## Specifications
The database will contains:
* Patient Name
  + First Name
  + Last Name
* Age
* Patient ID
* Test Results
    * Test name
    * Test Result
  1. HDL
  1. LDL
